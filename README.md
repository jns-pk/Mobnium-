# Mobnium-
🎵 A beautiful web-based harmonium you can play in your browser — tap on mobile or use keyboard shortcuts on PC.

## ✨ Features

- 🎹 **Two modes** — Mobile (big tap-friendly keys) and PC (classic piano layout)
- 🔊 **Realistic harmonium tone** — built with Web Audio API
- 🌀 **Reverb effect** — adds a natural room echo
- 🎛️ **Volume control** — adjustable slider
- 🎹 **MIDI support** — plug in a real MIDI keyboard and play
- ⌨️ **Keyboard shortcuts** — use A S D F G H J K L on your keyboard in PC mode
- 📱 **Auto-detects mobile or desktop** on load

## 🚀 How to Play

1. Open the link below in any browser
2. On **mobile** — tap the keys directly
3. On **PC** — press keyboard keys or click the piano keys
4. Use the switch button (top right) to toggle between Mobile and PC mode
